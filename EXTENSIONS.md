# VS Code Extensions

This file lists all extensions used in this setup.

You can install them **automatically using the script** or **manually with the command below**.

---

# Themes & Icons

- **Field Lights** – `sveggiani.vscode-field-lights`
- **Catppuccin Icons** – `catppuccin.catppuccin-vsc-icons`
- **Fluent Icons** – `miguelsolorio.fluent-icons`

---

# Core Editor

- **Vim** – `vscodevim.vim`
- **Vim Cheatsheet** – `andenetalexander.vim-cheatsheet`
- **Smooth Scroll** – `cyansprite.smoothscroll`
- **Error Lens** – `usernamehw.errorlens`
- **Git Graph** – `mhutchie.git-graph`

---

# Python Development

- **Python** – `ms-python.python`
- **Pylance** – `ms-python.vscode-pylance`
- **Python Env Manager** – `ms-python.vscode-python-envs`
- **Debugpy** – `ms-python.debugpy`
- **Black Formatter** – `ms-python.black-formatter`
- **PyGame Snippets** – `dhanush.pygame-snips`

---

# Web Development

- **HTML/CSS Support** – `ecmel.vscode-html-css`
- **Auto Rename Tag** – `formulahendry.auto-rename-tag`
- **Auto Open Markdown Preview** – `hnw.vscode-auto-open-markdown-preview`
- **Live Server** – `ritwickdey.liveserver`
- **HTML to CSS Autocomplete** – `solnurkarim.html-to-css-autocompletion`
- **Prettier** – `esbenp.prettier-vscode`
- **JavaScript Snippets** – `xabikos.javascriptsnippets`

---

# Utilities

- **Code Runner** – `formulahendry.code-runner`
- **Codeium** – `codeium.codeium`
- **Live Share** – `ms-vsliveshare.vsliveshare`
- **Spell Checker** – `streetsidesoftware.code-spell-checker`

---

# Screenshot / Visual Tools

- **Polacode** – `jeff-hykin.polacode-2019`
- **Colorize** – `kamikillerto.vscode-colorize`

---

# Manual Installation Command

If you prefer installing extensions manually, paste this command into your terminal:

```
code --install-extension sveggiani.vscode-field-lights --install-extension catppuccin.catppuccin-vsc-icons --install-extension miguelsolorio.fluent-icons --install-extension vscodevim.vim --install-extension andenetalexander.vim-cheatsheet --install-extension cyansprite.smoothscroll --install-extension usernamehw.errorlens --install-extension mhutchie.git-graph --install-extension ms-python.python --install-extension ms-python.vscode-pylance --install-extension ms-python.vscode-python-envs --install-extension ms-python.debugpy --install-extension ms-python.black-formatter --install-extension dhanush.pygame-snips --install-extension ecmel.vscode-html-css --install-extension formulahendry.auto-rename-tag --install-extension hnw.vscode-auto-open-markdown-preview --install-extension ritwickdey.liveserver --install-extension solnurkarim.html-to-css-autocompletion --install-extension esbenp.prettier-vscode --install-extension formulahendry.code-runner --install-extension codeium.codeium --install-extension xabikos.javascriptsnippets --install-extension ms-vsliveshare.vsliveshare --install-extension streetsidesoftware.code-spell-checker --install-extension jeff-hykin.polacode-2019 --install-extension kamikillerto.vscode-colorize
```
